package managers

bauh -  pacman, AUR,  flatpak, appimage GUI.

yay - CLI tool for installing AUR packages.

nix - package manager for reproducible configurations.

home arch install - install arch packages in your home directory, without root access.

debtap - repackage .deb packages for pacman.

snap - the use of snaps is widely controversial, primarily due to their centralized nature.

language specific - package / library installers for a specific language
