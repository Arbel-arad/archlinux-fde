LUKS2

LUKS2 encrypted volumes can be opened using multiple methods, i recommend using a TPM for convenience and either a FIDO2 key or recovery key as backups.

TPM2 unlock LUKS2

FIDO2  unlock LUKS2

clevis allows for more advanced options like network bindings.

 

detached LUKS headers allow you to keep the encryption header on another device, requiring that device to also be connected to perform decryption.
