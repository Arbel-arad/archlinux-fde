Getting Secrets to work on a WM (realogs.in)

add the following to your hyprland environment config:

export $(gnome-keyring-daemon --start --components=gpg,pkcs11,secrets,ssh)

edit the PAM configuration to unlock the keyring on login:

/etc/pam.d/login: (add the lines to the end of their corresponding section)

auth optional pam_gnome_keyring.so

session optional pam_gnome_keyring.so auto_star

/etc/pam.d/passwd:

password optional pam_gnome_keyring.so
