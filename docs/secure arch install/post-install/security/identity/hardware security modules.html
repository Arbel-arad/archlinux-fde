FIDO Alliance - Open Authentication Standards More Secure than Passwords

there are many manufacturers of security keys, each with their own different functionality and quality.

FIDO Certified Showcase (fidoalliance.org)

here are a few options:

  • yubico - yubico are widely known for their yubikeys, they have also contributed to many standards including webauthn and U2F.
  • google - google make some of the cheapest keys you can get that are probably not a scam.
  • onlykey - one of the few open source security keys.
  • authentrend - patented biometric authentication for FIDO2 keys, they also have FIPS-14o certification for most of their products.

 

remember to set the FIDO pin.
