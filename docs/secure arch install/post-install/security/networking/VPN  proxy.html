Category:Virtual Private Network - ArchWiki (archlinux.org)

 

TOR and  lokinet can be used to obfuscate network traffic sources and destinations to both darknet and clearnet sites.

i2p does the same, but doesn't have “exit nodes” so it can only access resources from it's own network.

proxychains allows you to “stack” multiple proxies in series.

 

personal VPNs:

  • tailscale / headscale - “overlay network” based on wireguard.
  • nebula - “mesh network” with for highly scalable environments.
