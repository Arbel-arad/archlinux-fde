network manager has some privacy issues by default.

  • WIFI passwords are stored in plaintext. enable wifi password encryption
  • MAC address leaks. enable mac address randomization / spoofing
  • hostname leaks to DHCP servers
  • machine-ID leaks
  • connectivity checks - system fingerprinting

 

the following config shows how to mitigate some of these issues:

NetworkManager/examples/nm-conf.d/30-anon.conf at main 路 NetworkManager/NetworkManager (github.com)

it will also enable the IPv6 privacy extensions.
