dns

when possible, it is always recommended to use encrypted DNS.

this can not only prevent outsiders from reading your queries, it also prevents some DNS spoofing attacks.

this can either be done with a local service like dnscrypt or using public servers directly with portmaster.

in more dangerous environments it is also possible to route DNS over TOR.
