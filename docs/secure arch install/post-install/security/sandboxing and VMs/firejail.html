warning! firejail has some problems working in FUSE mount paths.

fixes for firejail on FUSE / gocryptfs:

  1. the simplest solution is to enable the allow_other mount option, this will completely fix the issue but also reduce security as it allows the root user to directly access files in your home directory.
  2. the better option is to trick firejail out of scanning your home, so it won't panic. this is my chosen solution, but consider your own threat model before picking as it may cause issues, especially with default profiles.
