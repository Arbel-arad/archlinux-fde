sandboxing and VMs

10Sandboxing applications (archlinux.org)

firejail can be used together with apparmor to sandbox programs.

bubblewrap is used by flatpak to provide sandboxing.

qemu is a generic machine emulator and virtualizer.

libvirt is a collection of software that provides a way to manage virtual machines and other virtualization functionality.

containers can be used as a middle ground between application sandboxes and full virtualization, both in security and resources.

emulation - emulate other operating systems or architectures.
