identity

install and enable a secret-service so programs can securely store credentials.

enforce a password policy to ensure strong credentials are used - set minimum length and special characters.

use hardware security modules when possible, as they allow for the strongest authentication (keepassXC supports “virtual passkeys” if you don't have real ones).

use password managers for services that don't support security keys.

configure PAM to your liking (passwordless SUDO with FIDO keys, 2FA).

 

user certificates can be stored using SSH-agent and  GPG-agent for their respective types.
