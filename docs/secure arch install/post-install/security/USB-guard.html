there is also a fork of the USBGuard applet but the state of that is not clear.

 

install USBGuard and usbguard-notifier

Warning :Make sure to actually configure the daemon before starting/enabling it or all USB devices, including keyboard and mouse, will immediately be blocked!

 

create and install a ruleset based on currently connected devices:

Warning :connect any required devices (keyboard, mouse, security keys, network adapter) before running this command, or they will not connect anymore.

usbguard generate-policy > rules.conf
nano rules.conf
 #(review/modify the rule set)
sudo install -m 0600 -o root -g root rules.conf /etc/usbguard/rules.conf
sudo systemctl restart usbguard

allow the “users” group to get notifications:

sudo usbguard add-user -g users -d listen
