remote backup

syncthing is a serverless P2P file syncing system, it is very simple to set up and works well for small environments with a few devices.

nextcloud is a self-hosted cloud solution, similar to google workspace or microsoft 365. it includes file syncing, web editors, sharing and much more.

borgbackup is a deduplicating, compressed and encrypted backup solution.
